HDF5 MATLAB Examples

The examples below were created and tested with HDF5-1.8 using MATLAB R2009b, on win32 and glnxa64.

Please refer to the README.txt file for key points a user should consider while perusing these examples. For further information, see the Summary of HDF5 Capabilities in MATLAB.
